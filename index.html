MATLAB GUI Application: Additive Pixel Steganography

  • Developed a MATLAB-based GUI application for Additive Pixel Steganography, allowing users to encode text into both grayscale and color (RGB) images.

  • Implemented a custom algorithm that adds ASCII values to pixel data, ensuring compatibility with 8-bit images through overflow handling.

  • Designed a user-friendly interface for easy image upload, text input, and message extraction, demonstrating strong skills in image processing, GUI design, and data encoding techniques.

Multi-Client Chat Application: Java Swing & MySQL
